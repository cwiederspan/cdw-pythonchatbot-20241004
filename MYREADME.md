# My Steps for Creating

1. Use Teams Toolkit to create the application  
2. Make sure to create the python .venv  
3. Add the Azure OpenAI settings in the .env.testtool.user file  
4. Create the App Registration for -local  
5. Set the App Registration values in the .env.local and .env.local.user files  
6. Comment out the aadApp/create section in the teamsapp.local.yml file  
7. Update the end of the file where it's setting the env variables  
8. Create the App Registration for -dev  
9. Provision  
10. Deploy
